## 模型配置建议

### 单卡配置（推荐）

```bash
--max-model-len 8192          # 图像+文本
--tensor-parallel-size 1
--gpu-memory-utilization 0.85
```

### 高性能配置

```bash
--max-model-len 16384         # 支持更大图像/视频
--tensor-parallel-size 4      # 4卡并行
--gpu-memory-utilization 0.90
```

### 资源需求

| 配置 | 显存 | max_model_len |
|------|------|---------------|
| 单卡 | ~17GB | 8192 |
| 4卡 | ~4.5GB/卡 | 16384 |

---

## 使用建议

### 最佳实践

1. **图像理解**: 使用 512x512 或更小图片以获得最佳性能
2. **视频理解**: 建议使用 URL 方式而非 Base64
3. **文本对话**: 性能优秀，适合生产环境
4. **多轮对话**: 上下文理解准确

### 注意事项

1. **max_model_len**:
   - 图像: 4096 足够
   - 视频: 建议 16384 或更高

2. **图片格式**:
   - 支持: JPG, PNG, WebP
   - 建议: 512x512 或更小

3. **视频格式**:
   - 支持: MP4, AVI, WebM
   - 建议: 使用 URL 而非 Base64

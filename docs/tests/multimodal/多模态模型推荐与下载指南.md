# vLLM 多模态模型推荐与下载指南

## 推荐测试模型

根据 vLLM 官方文档和 Ascend NPU 兼容性，推荐以下模型进行测试：

---

## 一、图像理解模型 (推荐度 ⭐⭐⭐⭐⭐)

### 1. LLaVA-1.6-Next (推荐)

**特点**:
- 参数量: 7B, 13B, 34B
- 流行的开源视觉语言模型
- 高质量图像理解
- 支持中文

**下载命令**:
```bash
# 7B 模型 (推荐单卡测试)
modelscope download --model llava/llava-v1.6-vicuna-7b --local_dir ./llava-v1.6-7b

# 13B 模型 (推荐2-4卡)
modelscope download --model llava/llava-v1.6-vicuna-13b --local_dir ./llava-v1.6-13b

# 或使用 HuggingFace
huggingface-cli download llava-hf/llava-v1.6-vicuna-7b-hf --local-dir ./llava-v1.6-7b
```

**测试命令**:
```bash
docker run --rm \
    --device=/dev/davinci5:/dev/davinci0 \
    vllm-omni:v0.20.2rc \
    python3 -m vllm.entrypoints.openai.api_server \
        --model /models/llava-v1.6-7b \
        --port 8003
```

---

### 2. Qwen2-VL-7B (强烈推荐 ⭐⭐⭐⭐⭐)

**特点**:
- 参数量: 2B, 7B, 72B
- 阿里通义千问视觉模型
- 支持中文，效果优秀
- 支持图像和视频

**下载命令**:
```bash
# 7B 模型
modelscope download --model Qwen/Qwen2-VL-7B-Instruct --local_dir ./Qwen2-VL-7B-Instruct

# 2B 模型 (轻量级，适合快速测试)
modelscope download --model Qwen/Qwen2-VL-2B-Instruct --local_dir ./Qwen2-VL-2B-Instruct

# HuggingFace
huggingface-cli download Qwen/Qwen2-VL-7B-Instruct --local-dir ./Qwen2-VL-7B-Instruct
```

---

### 3. InternVL2-8B

**特点**:
- 参数量: 2B, 8B, 26B, 76B
- 书生视觉大模型
- 多模态能力全面
- 中文支持好

**下载命令**:
```bash
# 8B 模型
modelscope download --model OpenGVLab/InternVL2-8B --local_dir ./InternVL2-8B
```

## 八、注意事项

### 1. 模型路径

确保模型路径正确:
- ModelScope: `Qwen/Qwen2-VL-7B-Instruct`
- HuggingFace: `Qwen/Qwen2-VL-7B-Instruct`
- 本地: `/models/Qwen2-VL-7B-Instruct`

### 2. 显存配置

根据模型大小调整:
```bash
--gpu-memory-utilization 0.85  # 小模型
--gpu-memory-utilization 0.95  # 大模型
```

### 3. 多卡配置

```bash
# 2卡
--tensor-parallel-size 2

# 4卡
--tensor-parallel-size 4
```
